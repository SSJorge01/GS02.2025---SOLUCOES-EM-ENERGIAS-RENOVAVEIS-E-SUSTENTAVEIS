## Introdução

Um dos principais pontos esperados para o futuro do trabalho seria a transição para modelos de trabalho mais sustentáveis, o que implica em repensar como a energia é utilizada nos espaços produtivos. Para que isso aconteça, é necessário uma melhora na eficiência energética nos ambientes de trabalho, juntamente do uso de fontes renováveis.

## Conexão com o Futuro do Trabalho

Esse projeto visa promover um ambiente de trabalho mais eficiente e sustentável, se utilizando de técnicas de automação e análise de dados para melhorar a eficiência energética do local em questão, diminuindo os gastos de energia e como a mesma é gasta.

## Coleta e Análise de Dados

Para os dados de consumo/geração de energia utilizados nesse projeto, optamos por utilizar uma base de dados fictícia, simulando o consumo de energia de um pequeno espaço de trabalho.

A base em questão se encontra nesse repositório, sendo o arquivo com o nome de 'dados.csv'.

## Desenvolvimento da Solução

Como solução da problemática, escolhemos seguir na linha de raciocínio da opção A: "Análise de Dados", se utilizando de técnicas de modelagem computacional para analisar os dados de consumo da planta elétrica, a fim de verificar e recomendar meios para melhoras da eficiência energética.
